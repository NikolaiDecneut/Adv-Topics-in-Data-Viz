# Apex Legends Steam Player Chart

For this assignment, I created a basic D3 bar chart using an external CSV file. The chart shows average monthly Steam player counts for Apex Legends from December 2025 through April 2026.

## Files Included

- `index.html`
- `style.css`
- `main.js`
- `apex_players.csv`
- Screenshot of the final chart

## Data Source

The data comes from SteamCharts: https://steamcharts.com/app/1172470

## Description

This chart uses D3 to load data from an external CSV file, convert the player count values into numbers, create x and y scales, draw bars, and add axes.
